# Meditation app

## 🔨 作ったもの

作業や瞑想に使える環境音を再生するアプリケーションを構築する。
タイマーと 2 つのサウンドを選択することができる。

## 👊 使うもの

- Vanilla JS
- HTML
- CSS

## ✊ 得られるもの

JS を使って音と映像を扱うこと方法がわかる

## 👍 自分で追加した機能

- レスポンシブ対応
- ロゴ追加
- タイマーの 0 埋め
- 音量調整バー
